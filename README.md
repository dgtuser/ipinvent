# IP Invent

IPAM-подобное приложение для инвентаризации и хранения IP-адресов с авторизацией пользователей и SQLite.

## Что есть в проекте

- backend на `Python` без внешних зависимостей
- база данных `SQLite` в файле `ipinvent.db`
- первичная настройка с созданием первого администратора
- авторизация через сессию и роли `admin`, `operator`, `viewer`
- CRUD для IP-адресов
- CRUD для групп
- управление пользователями для администратора
- импорт CSV и экспорт текущей выборки
- поиск, фильтры, сортировка и аудит записей
- журнал последних действий

## Структура

- `server.py` — HTTP-сервер, API, авторизация и работа с SQLite
- `index.html` — интерфейс приложения
- `styles.css` — темная тема в стиле IPAM-панелей
- `app.js` — клиентская логика, работа с API, таблицами и модалками
- `ip-inventory-template.csv` — шаблон для импорта

## Запуск

1. Откройте терминал в папке проекта.
2. Выполните:

```powershell
py server.py
```

3. Откройте в браузере:

```text
http://127.0.0.1:8000
```

## Первый вход

Если пользователей еще нет, приложение автоматически покажет форму первичной настройки.

Там нужно:

1. создать первого администратора
2. войти в систему
3. при необходимости добавить остальных пользователей

## Роли

- `admin` — полный доступ, включая пользователей
- `operator` — работа с IP-адресами и группами
- `viewer` — только просмотр

## CSV

- импорт поддерживает `;` и `,`
- существующая запись обновляется, если IP уже есть в базе
- шаблон лежит в `ip-inventory-template.csv`

## Хранилище

- база создается автоматически: `ipinvent.db`
- временные артефакты Python попадают в `__pycache__`
